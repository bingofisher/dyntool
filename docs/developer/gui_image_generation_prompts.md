# GUI 示意图生成 Prompt

> **稳定性**：Private / implementation detail — 仅供内部参考，随 UI 变动更新。  
> 版本：2026-04-27  
> 每条 prompt 自包含，可直接粘贴使用，无需替换任何占位符。  
> Midjourney 末尾加 `--ar 16:9 --v 6 --style raw`；DALL-E 3 / 通义万象直接使用。

---

## 公共视觉风格说明（已内嵌在每条 prompt 中）

所有图均遵循：扁平浅色主题；背景 #F3F5F7；白色卡片 1px #D7DEE8 边框 8px 圆角；蓝色强调 #1F5AA6；绿色状态 #2E7D32；无阴影无渐变；中文 UI 文本；9pt 无衬线字体；Windows 11 桌面应用风格；专业数据分析工具质感。
