# GUI 视觉规格说明书

> 版本：2026-04-27  
> 目标分辨率：1366×768（最小）/ 1920×1080（标准）/ 2560×1440（2K 横屏）  
> 框架：PySide6 · QMainWindow + QDockWidget + QStackedWidget

---

## 一、空间语法（所有页面共享）

### 1.1 全局区域划分

```
┌──────────────────────────────────────────────────────────────────┐
│ 上下文条  32px  │ 项目名 · 主集名 · 状态文字          │ 进度条  │
├──────┬─────────────────────────────────────────────────────────  │
│      │ 总览  │  导入与筛选  │  数据处理  │  图形绘制  │  导出   │
│ 对   │───────────────────────────────────────────────────────── │
│ 象   │                                                          │
│ 树   │                  页面内容区                              │
│      │              （每页独立布局）                            │
│ dock │                                                          │
├──────┴──────────────────────────────────────────────────────────┤
│ 底部任务面板  96px（收起）/ 220px（展开）                       │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 固定尺寸规则（跨分辨率）

| 区域 | 1366px 宽 | 1920px 宽 | 2560px 宽 |
|---|---|---|---|
| 上下文条高 | 32px | 32px | 32px |
| 导航栏高 | 30px | 30px | 30px |
| 对象树 dock 宽 | 170px | 175px | 180px |
| 底部面板高（收起） | 84px | 96px | 96px |
| 底部面板高（展开） | 220px | 240px | 260px |
| 页面左侧配置栏 | ≤300px | ≤320px | ≤340px |
| 页面分隔线 | 5px | 5px | 5px |

### 1.3 页面内部两栏语法

所有非总览页遵循：

```
┌─ 左侧配置栏（固定宽，可拖拽）─┬─ 右侧主工作区（弹性，占满剩余）─┐
│  参数组 A                      │                                  │
│  参数组 B                      │    主内容（表格 / 画布 / 卡片）  │
│  ────────────────              │                                  │
│  [主动作按钮]                  │                                  │
└────────────────────────────────┴──────────────────────────────────┘
```

左侧配置栏：
- 最小宽度 200px，最大宽度 340px，默认 280px
- 内部用 QScrollArea，超出时纵向滚动，不横向滚动
- 底部 sticky：主动作按钮始终可见，不随内容滚动

## 八、窗口尺寸变化规则

### 8.1 拉伸行为

```
【宽度拉伸（用户拖拽窗口右边缘）】

1366px → 1920px → 2560px

变化的：
  ✓ 右侧主区（弹性伸展，吃掉所有增量）
  ✓ 导航 Tab 间距（等比例扩散）

不变的：
  ✗ 对象树宽（钉死 170-180px）
  ✗ 左侧配置栏宽（用户拖拽值，不自动变）
  ✗ 上下文条高（32px）
  ✗ 导航栏高（30px）
  ✗ 底部面板高（收起 96px，展开 240px）
```

### 8.2 压缩行为（窗口缩小）

```
【宽度 < 1100px：极限压缩模式】

对象树 Dock 自动折叠（变为侧边图标，点击弹出）：
┌──┬─────────────────────────────────────────┐
│≡ │ 总览 │ 导入筛选 │ 处理 │ 绘图 │ 导出   │

右侧主区：最小 400px，不再缩小（整窗口出现横向滚动条）

【宽度 < 900px：不支持，提示用户】
弹出提示条（非对话框）：
```

## 九、页面跳转与状态保持规则

### 9.1 跳转时的状态保持

```
【切换页面时，每个页面的局部状态保持不变】

例：用户在数据处理页选择了"峰值提取"并展开了参数
→ 切换到图形绘制页查看图形
→ 切换回数据处理页
→ 仍显示"峰值提取"和已填的参数，不重置

实现方式：局部状态存在各 Workspace Widget 的成员变量中，
session.current_module 切换只控制 StackedWidget 显示哪个 widget，
不触发子 widget 重建。
```

### 9.2 跨页范围联动

```
【范围切换的传播】

触发点（任意一处）：
  ① 对象树点击子集
  ② 处理页左侧范围下拉框选择
  ③ 绘图页左侧范围下拉框选择

传播规则：
  → session.current_scope 更新（单一数据源）
  → 当前可见页面的范围标签即时刷新
  → 其他页面的范围标签在下次显示时刷新（懒更新）
  → 对象树高亮联动到对应行

不传播的：
  × 不强制切换页面
  × 不清空右侧已有的执行结果（用户手动刷新）
```

### 9.3 任务运行时的跳转限制

```
【后台任务运行中】

允许的操作：
  ✓ 切换到其他页面查看
  ✓ 在对象树中浏览
  ✓ 展开/收起底部任务面板

禁用的操作（控件变灰 + tooltip 说明原因）：
  ✗ 导航栏"导入与筛选"Tab（正在导入时）
  ✗ 左侧"执行"按钮
  ✗ 菜单栏"新建/打开项目"

禁用指示：
  Tab 文字变 #C9D3E0
  鼠标悬停 tooltip："当前有任务正在运行，请等待完成"
  不弹对话框，不阻断查看
```
