# 自动模块参考

稳定性：`Internal API`

本分区提供基于 `mkdocstrings` 自动生成的模块参考，用于补充手写的使用说明页。

## 怎么用

- 先看 [公开 API](../api/public_api.md)
- 维护或扩展时再看 [内部代码说明](../api/internal_api.md)
- 需要精确签名、字段和 docstring 时进入自动模块参考

## 推荐阅读顺序

1. `dyntool`
2. `dyntool.storage`、`dyntool.plotting`、`dyntool.logging`
3. `dyntool.domain.models`、`dyntool.domain.samples`
4. `dyntool.application.runtime_binding`、`dyntool.domain.runtime`
